Integrating SciTE with Ren'Py

With the Ren'Py distribution, we ship versions of SciTE for Windows and Linux x86. This version of SciTE includes Ren'Py support, and is configured with settings that are appropriate for editing Ren'Py.

From Ren'Py, SciTE can be invoked on the current line by pressing shift-e.

From SciTE, Ren'Py can be started by pressing ctrl-0.
