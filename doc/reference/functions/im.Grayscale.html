im.Grayscale

Function: im.Grayscale (im, desat=(0.2126, 0.7152, 0.0722)):

This image operator converts the given image to grayscale. desat is as for im.matrix.saturation.


Example

init:
    image city gray = im.Grayscale("city.jpg")
