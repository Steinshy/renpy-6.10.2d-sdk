renpy.has_label

Function: renpy.has_label (label):

This function returns true if a label exists in the game script, and false otherwise.

Example

if renpy.has_label('episode_3'):
    jump episode_3
else:
    "To Be Continued..."
    jump game_end

label episode_3:
    "It was a dark and stormy night..."
