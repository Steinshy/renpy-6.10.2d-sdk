ui.at

Function: ui.at (position):

The ui.at function takes a position or motion, and applies it to the next widget created using a ui function.

Example

# This scrolls the button in from the left side of the screen.

$ ui.at(Move((0.0, 0.5, 1.0, 0.5), (0.5, 0.5, 0.5, 0.5), 1.0))
$ ui.textbutton("Click Me!", clicked=ui.returns(True))
$ ui.interact()
