ui.clear

Function: ui.clear ():

Clears the current layer of widgets. Not particularly useful in practice, and basically duplicative of renpy.scene.


Example

# Now, you can do renpy.scene() in 3 lines instead of one!
ui.layer("master")
ui.clear()
ui.close()
