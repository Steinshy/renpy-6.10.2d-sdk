ui.frame

Function: ui.frame (**properties):

A frame contains a single widget. This is similar to a window, but may be styled differently.

Example

python:
    # Stats Window
    ui.frame(xpos=0,
             ypos=0,
             xanchor='left',
             yanchor='top',
             xfill=True,
             )

    ui.vbox()

    ui.text('Statistics')
    ui.null(height=20)

    for name, range, value in stats:
        ui.hbox()
        ui.text(name, minwidth=150)
        ui.bar(range, value, ypos=0.5, yanchor='center')
        ui.close() # for the ui.hbox

    ui.close() # for the ui.vbox
