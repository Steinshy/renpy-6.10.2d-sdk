ui.window

Function: ui.window (**properties):

A window contains a single widget. It draws that window atop a background and with appropriate amounts of margin and padding, taken from the window properties supplied to this call. The next widget created is added to this window.

Example

python:
    # Single-child widget, contains the following ui.vbox
    ui.window()

    # Multiple-child widget - all new widgets following this will be
    # added as children to it, until the next ui.close
    ui.vbox()
    ui.text("To get to the next screen, click the 'Continue' button.")
    ui.close()
